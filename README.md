# IT Infrastructure: Standard Operating Procedures (SOPs)

##  Project Overview
This repository serves as a technical knowledge base for Helpdesk and Desktop Support operations. It contains professional-grade documentation designed to reduce MTTR (Mean Time to Resolution) and ensure high security standards across a distributed workforce.

##  Skills Demonstrated
* **Technical Writing:** Creating clear, step-by-step instructions for complex IT tasks.
* **Security Governance:** Integrating BitLocker, MFA, and BIOS security into standard workflows.
* **Troubleshooting Logic:** Defining clear escalation paths and diagnostic steps.
